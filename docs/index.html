os-specific-runner

A GitHub Action that runs a different shell command per operating system —
Linux, macOS, Windows, AIX, FreeBSD, OpenBSD, and SunOS.

Read the Docs
🐧 🍎 🪟

Multi-OS support

Target Linux, macOS, and Windows in a single workflow step, plus AIX, FreeBSD, OpenBSD, and SunOS.

🐚

Configurable shells

Choose bash, zsh, sh, pwsh, cmd, or python independently per platform.

📁

Working directory

Control where the temporary script file is written with the working_directory input.

⚡

Zero dependencies

Pure JavaScript action — fast startup, no Docker pull, and no extra tools required in your runner.

Quick Start

- name: Run OS-specific command
  uses: carlkidcrypto/os-specific-runner@v2
  with:
    linux:   echo "Hello from Linux"
    macos:   echo "Hello from macOS"
    windows: echo "Hello from Windows"
